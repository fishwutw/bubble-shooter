# Bubble Flow 1.6.3 穩定與手感調整版

這版依照你的回饋調整三個重點：

## 1. 更穩定

- 遊戲開始後會鎖定盤面幾何資料。
- iPhone Safari 網址列或工具列高度微變時，只更新 canvas，不重新排列上方泡泡。
- 只有真正改變螢幕寬度或轉向時，才會重建盤面。

## 2. 發射速度加快

- 發射速度從約 `Math.max(7.3, W / 78)` 提升到 `Math.max(9.2, W / 62)`。
- 手感會更俐落，等待泡泡飛行的時間更短。

## 3. 上方新增排往下移更慢

- 新增排的垂直下移動畫從 0.22 秒加長到 0.46 秒。
- 仍然只做垂直移動，不做左右補間。

## 4. 輔助線對準發射區圓心

- 輔助線起點改為發射泡泡的圓心。
- 瞄準角度也改以圓心計算。
- 發射泡泡也從圓心出發，視覺與軌跡會更一致。

## 使用方式

把 `index.html` 覆蓋 GitHub Pages repository 原本的檔案。

建議用以下網址避開快取：
