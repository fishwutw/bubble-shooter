# Bubble Flow 1.6.1 修正版

這版先針對你回報的兩個問題修正：

## 修正 1：新增上方氣泡時畫面跳動、顏色或位置改變

原因主要有兩個：

1. iPhone Safari 的網址列收合會觸發 `visualViewport.resize`，舊版會因此重算甚至 reset 盤面。
2. 新增排時舊版一邊讀取 `board`、一邊覆寫 `board`，視覺上容易產生位置瞬變。

本版修正：

- 不再讓 `visualViewport.resize` 直接重算遊戲盤面。
- 新增排時改用 `nextBoard` 建立新盤面，保留每顆泡泡的 color，不重抽、不改色。
- 新增排時加入 0.24 秒平滑下移動畫。

## 修正 2：消除左側氣泡，右側氣泡也跟著掉落

原因是舊版初始盤面與新增排可能已經產生「預先懸空」的泡泡島，只是前面沒有被清掉；當任一側發生消除時，掉落判定會把這些早就懸空的泡泡一起判定為掉落。

本版修正：

- 初始盤面前四排保持完整連接，第五排才留少量空洞。
- 新增排前會靜默清理預先懸空的泡泡。
- 掉落判定只在消除後執行，降低非相關區域誤掉的機會。
