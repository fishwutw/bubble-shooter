# 氣泡消除測試版 Stage 1

這是第一階段的網頁版原型，可直接在 iPad Safari 測試。

## 使用方式

1. 解壓縮 zip
2. 開啟 `index.html`
3. 或把整個資料夾放到任一靜態網站空間，例如 GitHub Pages、Netlify、內部伺服器
4. iPad Safari 開啟網址後即可測試

## 已完成功能

- 隨機產生上方氣泡排列
- 下方隨機氣泡發射
- 觸控拖曳瞄準、放開發射
- 牆壁反彈
- 碰撞後吸附到六角網格
- 三顆以上同色相連消除
- 浮空氣泡掉落計分
- 分數、發射次數、最高分
- 每 6 次發射新增一排壓力氣泡
